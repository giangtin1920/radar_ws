Version 3.1.0 ============= Overview ======== This software package demonstrates the use of TI mmWave sensors to count and track multiple vehicles in a traffic monitoring application. Detection and tracking algorithms run onboard the device to localize vehicles and track their movement. * Supported EVM platforms: * IWR1843BOOST * IWR6843ISK (ES2.0) * mmWave SDK 3.x architecture compliant * Supports 18xx and 68xx devices * 3D pt cloud processing and tracking enabled * Visualizer with data logging and playback features New and Updated Features ======================== * Updated to support SDK 3.5.x Known Issues & Work Arounds ============ * An unresolved UART issue can result in missing bytes in packets. The work around used in this release is to introduce 1msec delays following UART writes. This impacts the maximum achievable frame rate due to increased processing time. Limitations ========================= * The intended use of this traffic monitoring lab is to monitor traffic lanes along the boresight of the sensor. Due to the initial assumptions of the tracking algorithm, tracking of cross traffic across the azimuth field of view will be limited. --------------------------------- Release Notes Archive ========================= Version 3.0.1 ----------------- New and Updated Features * Updated 18xx MSS projectspec with xwr18xx radarss firmware and 18xx pre-compiled binary.
