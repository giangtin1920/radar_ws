Multiple Gesture and Motion Detection Version 2.1.1 ============= Overview ======== This is the Multiple Gesture and Motion Detecion lab for the TI IWR68xx EVM. Features ======== The following features are supported: * Detection and classification of two different hand gestures * Person/motion detection at different ranges up to 3.5m * Supports mmWave SDK 3.5.0.x * Supports IWR6843 AOP ES2.0 devices. * Supports IWR6843 ODS ES2.0 devices. New and Updated Features ======================== * Migrated support for IWR6843AOP EVM ES1.0 to ES2.0 * Reoriented AOP gesture so EVM is not rotated.
