Version 1.0.0 ============= Overview ======== This is the mmWave Low Power Demo for the xWR6843 and xWR6443 devices. This lab is a clone of the standard 64xx Out of Box Demo with the addition of several low-power functions to put the device into idle mode. Please see the user guide for more information on these functions and how to integrate them in both new and existing projects.
