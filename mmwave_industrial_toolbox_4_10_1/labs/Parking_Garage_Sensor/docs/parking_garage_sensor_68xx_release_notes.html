Version 1.0.0.0 ============= Overview ======== This lab demonstrates the use of TI mmWave sensors for Parking Garage occupancy applications. The unique range, velocity, and angle data from mmWave sensors can enable the detection of objects or cars as they enter or exit zones of interest. * SDK Version: 3.5.0.4 * Supported EVM platforms: * IWR6843ISK (ES2.0) New and Updated Features ======================== * Initial Release Known Issues & Work Arounds ============ * An unresolved UART issue can result in missing bytes in packets. The work around used in this release is to check and skip parsing of frame with missing bytes in gui. Limitations ========================= * The minimum range of the static object detection is currently limited by the occurence of false detections. --------------------------------- Release Notes Archive =========================
