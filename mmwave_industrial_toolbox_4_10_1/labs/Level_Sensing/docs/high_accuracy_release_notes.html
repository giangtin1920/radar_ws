Version 3.3.0 ============= Whats New? ======================== * The 68xx and 14xx verions of this lab now include a multi-peak detection detection feature which performs the Zoom FFT on three distinct objects in the scene. * The [High Accuracy Visualizer](https://dev.ti.com/gallery/view/4768107/High_Accuracy_Visualizer/) has been updated with color indicators and also compatibility for both the 68xx and the 14xx Multi-Peak Versions. Overview ======== This lab demonstrates the use of TI mmWave sensors for applications requiring a high degree of range accuracy. Using the Zoom-FFT technique one can achieve range accuracy on the order of millimeters. * Supported EVM platforms: * IWR1443BOOST (Compliant with SDK 2.1) * Featuring Multi-Peak Detection on 14xx version of lab * IWR1642BOOST (Compliant with SDK 2.1) * IWR6843ISK (ES2.0) (Compliant with SDK 3.3) ** (Both 68xx and 64xx versions) ** * Featuring Multi-Peak Detection on 68xx version of lab * IWR6843ODS (ES2.0) (Compliant with SDK 3.3) ** (Both 68xx and 64xx versions) ** * Featuring Multi-Peak Detection on 68xx version of lab * IWR6843AOP (Compliant with SDK 3.2 AOP) ** (Both 68xx and 64xx versions) ** * Featuring Multi-Peak Detection on 68xx version of lab Features ======================== * mmWave SDK 3.x compliant for IWR6843 Versions * mmWave SDK 2.1 compliant for IWR1443/1642 Versions ---------------------------------
