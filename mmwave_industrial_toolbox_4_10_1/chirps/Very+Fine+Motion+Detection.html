Very Fine Motion Detection

Introduction

This use case provides the mmWave sensor configuration for detecting fine motion in a conference room. Fine motion in this context refers to movements such as human breathing, typing, moving their arm etc. This configuration configures the mmWave sensor for very high velocity resolution using which we can detect the types of motion defined above. This particular configuration has a maximum range of about 6 meters and the sensitivity of the detected motion varies with distance from the sensor. The detection field of view depends upon the design of the antenna. This configuration is based on the corresponding mmWave experiment called Very Fine Motion Detection vs PIR and the reader is encouraged to visit the experiment page for more details.

Testing and Results

The chirp configuration files provided above were tested with the corresponding EVMs (IWR443BOOST, IWR1642BOOST, and IWR6843 ISK) using the out of box demo and mmWave Demo Visualizer. The motion was detected on the X-Y Scatter and Doppler-Range plots on the mmWave Demo Visualizer. Please refer to the Very Fine Motion Detection vs PIR experiment for details of the Test Setup and Results.
