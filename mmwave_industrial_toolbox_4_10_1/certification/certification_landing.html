# Overview TI mmWave Certification Database is a repository of the certification documents, test reports and other collaterals relating to various requlatory compliance certification done on the mmWave radar evaluation modules. [TI mmWave Radar Device Regulatory Compliance Guide](https://www.ti.com/lit/an/spracp3c/spracp3c.pdf) appnote is available on TI.com to provide some technical insights on the test standards. # Content [[y! Reference Material Please note that the content of this Database are for reference only ]] * IWR6843AOPEVM - RED certification collateral - FCC certification collateral * IWR6843ISK - RED certification collateral - FCC certification collateral * IWR6843ISK-ODS - RED certification collateral * IWR1843BOOST - RED certification collateral * RTRIM Adjustment Tool - Includes User Guide, prebuilt binary, and source code # Version History ## Changes in Version 4.9.0 The following is a list of changes compared to the previous release. * IWR6843AOPEVM now includes FCC certification collateral * IWR6843ISK now includes FCC certification collateral * IWR1843BOOST has been added with RED certification collateral * RTRIM Modification Guide has been updated with APLL RTRIM adjustment method using an mmWave-DFP API * Fixed an issue where IWR6843AOPEVM Radio Equipment Directive Test Plan would not appear directly ## Changes in Version 4.6.0 The following is a list of changes compared to the previous release. * IWR6843AOPEVM folder updated to contain test reports for IWR6843AOP ES2. * IWR6843ISK-ODS folder updated to account for changes to mean power interpretion by removing duty cycle correction factor from fundamental and OOB measurements.
