Response of Radar to Rain

Introduction

The objective of this experiment is to determine the impact of rain on the ability of TI mmWave Radar to sense the desired object. For the purpose of this experiment, we simulated two mock rainy situations, one with a sprinkler representing light rain and the other with a garden hose representing medium to heavy rain. The experiment was repeated using two different TI mmWave platforms: IWR1443 and IWR6843 both of which showed similar performance.

Setup

The setup for this experiment can be divided into two parts namely Physical setup and Software setup. These are detailed below:

Physical Setup

The Physical setup includes information about the scene such as the distance between the object and the sensor antenna, the type of object used, environmental conditions of interest (dry vs rainy), hardware setup etc. The physical setup used for this experiment is given below:

  1. A corner reflector with RCS of 0.5m2 was used as a target object and it was kept 8-10 meters from the EVM, with the corner reflector facing the sensor antenna.
  2. The corner reflector was mounted on a tripod and the scene was setup in the backyard of a home.
  3. The hardware setup included an IWR1443 EVM or an IWR6843 EVM, connected to a Windows Host PC.

  4. Rain was simulated using a sprinkler or a garden hose with a nozzle.

Software setup

  1. The experiment was repeated on two different EVMs: IWR1443 and IWR6843. The corresponding mmWave SDK Out-of-Box example was used on both platforms.
  2. The Windows host PC was running the mmWave Demo Visualizer to show the range and scatter plots. The mmWave Demo Visualizer was configured for Best Range as shown in the setting below.

Method

The steps presented below were repeated for both Dry and Rainy conditions.

  1. Create the physical setup as per the details provided above.
  2. Flash the mmWave SDK Out-of-Box Demo on the mmWave EVM and connect the EVM to the host PC.
  3. Launch mmWave Demo Visualizer on the host PC and setup the Visualizer as per the configuration shown below:
    • Platform - xWR14xx, xWR68xx
    • Desired Configuration - Best Range
    • Maximum Unambiguous Range - 30m
    • Range Detection Threshold - 15 dB
  4. Press Send config to MMWave Device
  5. Select the Plots Tab to look at the Scatter Plot and the Range Profile plot
  6. It is recommended to first run the experiment in Dry condition to ensure that the object can be detected in Dry condition. This serves as the control against which the effects of Rain can be compared.
  7. Turn on the Sprinkler placed near the corner reflector to generate rainy conditions and repeat the experiment

Results

The steps presented above were repeated for Dry, Light rain and Medium rain conditions. The Scatter Plots and Range Profile Plots were used to determine if the object could be detected and the results are summarized below:

  1. The object was detected in both dry and rainy conditions.

  2. Rain on the antenna radome, and on objects in the static scene can cause diffraction, which lowers the peak value, and with multiple objects lowers the SNR.

Conclusion

  1. The mmWave radar sensor was able to detect the object in a static scene in both dry and rainy conditions.
  2. In rain, a 1-2 db loss in peak magnitude can be noticed, along with rain-deflections of objects in the scene (this is seen as a lowering of SNR). Therefore in rainy conditions, the object detection SNR (Range detection threshold in the mmWave Demo Visualizer) may need adjustment to detect the desired objects.
